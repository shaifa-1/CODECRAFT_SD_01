# CODECRAFT_SD_01 - Contact Management System

## Project Description
This is a GUI-based Contact Management System developed using Java and Spring Boot.
The application allows users to manage contacts efficiently with full CRUD  operations.

## Features
- Add new contacts
- View all contacts
- Update existing contact details
- Delete contacts
- User-friendly GUI interface

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML, CSS
- Eclipse IDE
- Git & GitHub

## Architecture
- Controller Layer
- Service Layer
- Repository Layer
- Model Layer

## How to Run
1. Import the project into eclipse
2. Configure MySQL database
3. Update application.properties
4. Run the spring boot application
